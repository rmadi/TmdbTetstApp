# TMDB Movie App (RNE UI, RN 0.80)

- UI: @rneui/base + @rneui/themed (no NativeBase)
- State: Redux Toolkit + RTK Query + redux-persist
- i18n: i18n-js (EN/HI)
- Config: react-native-config
- Aliases: babel module-resolver (no relative paths)

## Env
Copy `.env.example` to `.env` and set `TMDB_API_KEY`.
